# Horizon

**数字化转型 × AI 转型服务商**
Digital & AI Transformation Services

</div>

---

Horizon 帮助流程严谨的传统企业（化工、制造等流程工业）完成数字化与 AI 转型：

- 🏭 **业务数字化**：ERP 本地化与集成、移动审批、跨境网络、数据治理
- 🤖 **智能体员工（AI Agent Workforce）**：基于开源平台 [Multica](https://multica.ai/) 交付可派单、可审计的"数字员工"
- 🛡️ **安全闸门体系**：面向高合规行业的智能体治理框架——*智能体只提案，人来批准，网关强制执行，监察员事后对账*
- 📐 **AI 产研范式**：把工程经验沉淀为可复用的技能与门禁，让 N 个智能体复利式成长

## 方法论一瞥

| 原则 | 含义 |
|---|---|
| 本质安全 | 危险动作不是"禁止做"，而是通路不存在 |
| 纵深防御 | 平台/运行时/网关/流程/审计五层独立保护 |
| Fail-closed | 任何闸门组件异常即拒绝，绝不降级放行 |
| 四权分立 | 提案者 ≠ 批准者 ≠ 执行者 ≠ 监察者 |
